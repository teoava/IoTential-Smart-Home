# IoTential - Smart Home Automation Prototype

IoTential is a functional IoT concept designed to demonstrate the potential of automation in daily household activities. The project focuses on improving comfort, protection, and hygiene through sensor-based interactions.

## 🚀 Features
- **Smart Rain Protection:** Automatically detects rain using a moisture sensor and moves a servo-controlled arm to shield laundry.
- **Touchless Waste Management:** A motion-activated trash can that opens its lid when a user is detected within 25cm using an ultrasonic sensor.

## 🛠️ Technology Stack
- **Languages:** C++ (Arduino Sketch)
- **Hardware:** Arduino Uno, Servo Motors, HC-SR04 Ultrasonic Sensor, Rain/Moisture Sensor.
- **IDE:** Arduino IDE / VS Code.

## 📈 Challenges Overcome
Managed power distribution for multiple servos and resolved COM port conflicts by optimizing the serial communication baud rate and modularizing the code logic.
